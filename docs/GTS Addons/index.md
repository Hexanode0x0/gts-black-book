# GTS Addon Collections

GTS addon collections are special mod collections on Nexus Mods designed specifically to work with GTS. This is about all they have in common.
How they function, how easy they're to install, or their content can vary vastly.

Since GTS itself is an active project, any addons may need to be updated to support newer versions, especially in wake of any large updates.
This page is used to keep track of what collections are currently up-to-date and supported.


!!! warning "NSFW"

    This page contains links to NSFW addons too, though no NSFW content is included on the page. All NSFW links are marked as such.
    
    Viewer discretion is advised when clicking on links.

    ??? note "Note on NSFW and Nexus Mods"

        Nexus' adult content flag is very broad and encompasses a lot of different things.

        On top of that, if any adult flagged mod is in a collection, then the whole collection is flagged adult.

        As of right now, NSFW in the context of this page refers to sexual content only. If any objectionable, non-sexual addons come about, this tag may broaden.
