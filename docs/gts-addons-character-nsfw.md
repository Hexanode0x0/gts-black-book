# Character & NSFW Addons

!!! warning "NSFW"

    This page contains links to NSFW addons, though no NSFW content is included on the page. All NSFW addons are marked as such.
    
    Viewer discretion is advised when clicking on links.

    ??? note "Note on NSFW and Nexus Mods"

        Nexus' adult content flag is very broad and encompasses a lot of different things. If a single adult flagged mod is in a collection, the whole collection is considered adult, even if it doesn't make use of any adult features of the mod.

        As of right now, NSFW in the context of this page refers to sexual content only. If any objectionable, non-sexual addons come about, this tag may broaden.
